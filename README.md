# 🚀 Engineering Dashboard - React State Management

Ce projet a été réalisé dans le cadre du **TP 2 : Manipulation de l'état et des hooks**. L'objectif était de concevoir une interface modulaire démontrant la maîtrise des Hooks fondamentaux de React (`useState`, `useEffect`) ainsi que la gestion des événements et des formulaires.

---

## 🛠️ Stack Technique
* **Framework :** React 18 (Functional Components)
* **Stylisation :** CSS3 Moderne (Architecture Grid & Flexbox)
* **Design System :** Glassmorphism & Minimalist UI
* **Hooks :** useState, useEffect

---

## 📂 Architecture du Projet
Le projet est segmenté en composants atomiques pour une meilleure maintenabilité :

* **`Compteur.js`** : Gestion d'un état numérique avec logique d'incrémentation.
* **`MessageBouton.js`** : Manipulation de chaînes de caractères via le State.
* **`FormulaireNom.js`** : Contrôle des entrées utilisateurs (Controlled Components) et gestion du cycle de soumission.
* **`CompteurEffet.js`** : Démonstration du cycle de vie et des effets de bord avec `useEffect`.

---

## 🎨 Design & Expérience Utilisateur
L'interface a été conçue pour dépasser le cadre académique et proposer une expérience typée **"SaaS Dashboard"** :
* **Typographie :** Utilisation de la police *Inter* pour une lisibilité maximale.
* **Interactivité :** Micro-interactions au survol des cartes (Hover effects).
* **Responsive :** Grille adaptative (Grid Layout) compatible mobiles et tablettes.

---

## 🚀 Installation et Lancement

1.  **Cloner le projet ou extraire l'archive :**
    ```bash
    cd tp2-state
    ```

2.  **Installer les dépendances :**
    ```bash
    npm install
    ```

3.  **Lancer l'application en mode développement :**
    ```bash
    npm start
    ```
    L'application sera accessible sur `http://localhost:3000`.
